プライバシーポリシー

本プライバシーポリシーは、「テーマ検索くん（nihuBridge デモ）」において提供される検索機能（以下「本サービス」）における、利用者情報の取扱いについて定めるものです。

取得する情報

本サービスでは、以下の情報を取得する場合があります。

  • 利用者が入力した検索クエリ（自然文またはキーワード）
  • API リクエストの技術的情報（IPアドレス、リクエスト時刻、エラー情報等）

これらの情報には、氏名、所属、メールアドレス等の個人を直接特定する情報は含まれません。

利用目的

取得した情報は、以下の目的にのみ利用します。

  • 検索機能の提供および正常な動作の確認
  • 不具合発生時の原因調査および改善
  • 不正利用防止およびシステム運用上の安全確保

これらの目的以外で利用することはありません。

外部サービスへの送信

本サービスでは、検索機能の実現のため、利用者が入力した検索内容を nihuBridge API に送信します。送信される情報は検索処理に必要な範囲に限定され、他の目的で利用されることはありません。

ログの保存期間

取得したログ情報は、運用および障害対応に必要な期間のみ保存します。原則として、保存期間は最大 30 日間とし、期間経過後は速やかに削除します。

第三者提供

取得した情報を、法令に基づく場合を除き、第三者に提供することはありません。

セキュリティ

本サービスでは、取得した情報への不正アクセス、漏えい、改ざん等を防止するため、適切な技術的および運用上の安全対策を講じます。

プライバシーポリシーの変更

本プライバシーポリシーの内容は、必要に応じて変更されることがあります。変更後の内容は、本サービス上に掲載した時点で効力を生じます。
